class  Hud
 Holds and displays the various hud objects (text and images) that are specified in the settingsFile. All the objects do not have an "isBeside" attribute are placed to corner of the specified partition. All objects are placed beside other objects by specifying the partition and the "isBeside" attribute to point to the id of the target object that you want the current object to be placed beside. Objects in the TopLeft and BottomLeft are placed from left to right. Objects in the TopRight and BottomRight are placed from right to left. More...
